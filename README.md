# ChengduBusLCD
## Python程序，用于模拟成都公交车内电显（目前应该不考虑做其他城市的）
**一、产品亮点**
  1.自动快速绘制，告别传统的手工绘制
  2.强大的容错机制，一般情况下软件稳定运行，不崩溃
  3.界面友好，信息提示多
  4.可支持自定义风格
**二、开发灵感**
B站车迷@成都地铁S11德阳线（UID：3546630506678721）原创了成都公交LCD模拟PPT，界面美观，易于上手，还原度高。但是，公交，无论是线路数量还是站点数量，都远远超过地铁，手工绘制工作量极大。且公交线路经常改线、站点经常改名，这无疑是雪上加霜。并且，公交线路的车内LCD屏同质化严重，主城区几乎全是这个风格，也没有识别色的区分。因此，采用程序代替我们完成这种批量专一的工作，将大大节省时间。这样，广大公交车迷在拍摄POV之后剪辑视频时，可以录制此软件展示界面，增加同步当前站点信息，给视频一种更高大上的感觉（up自卖自夸）。
**三、运行说明**
此为纯文字版，图文版请见docx文档。
本软件为压缩包形式绿色版，无需安装，在想要运行的位置解压。由于up主受到了之前成都地铁LCD模拟PPT被盗用的教训（还不止一次，不得不说车迷圈子还是有点乱），因此我再三考虑，使用了身份验证的方式，这样可以避免随意传播，我就能通过工坊后台或github后台了解哪些用户下载了本程序，发生滥用情况就能更方便查人。（就好比进入森林需要做防火身份登记一个道理）
不过，一步一步来，通过验证是相当简单的。这里以高清版的为例：
1.	右键选中下载好的zip文件，选择“解压到当前文件夹”。
2.	如无此命令，说明没有安装解压软件，个人推荐7-zip、Bandizip。
3.	双击后缀名为exe程序运行（请勿未解压直接运行，可能产生难以预料的行为）。
4.	首次运行会弹出验证，请复制此用户码，打开哔哩哔哩客户端，搜索用户“逐梦桃花源”进入私信界面，或者点击我的-工坊，点击右上角头像，点击我的订单，点联系创作者。将用户码发送给作者。
5.	作者将会在1天之内将密钥私信给您。请按提示复制到软件文本框，即激活成功。你会发现运行目录多了一个文件license.dat，不用管它。
6.	再次运行，即进入软件主界面。界面已经很详细，相信有手就会。请注意遵守设置面板下方的使用规范。
7.	如遇问题，请百度、AI或联系创作者，请不要随意给出差评。
**四、常见问题解答**
Q：怎么快捷获取站点纯文本文件？
A：
1.	电脑浏览器进入高德地图官网。
2.	搜索你想展示的线路（如70路）。
3.	选择你想要的运行方向，并展开站点详情。
4.	键盘Ctrl+all全选，复制到新建文本文档。
5.	删掉非站点内容，保存即可。注意最下面“成都星辰巴士客运公司”不是站点！（我自己内测的时候一直都没发现哈哈哈嗝）

Q：无法运行或报错。
A：（1）如果报错的是“dll初始化异常，应用程序无法正常启动”之类的提示，请百度搜索“dll修复工具”，看是否能解决问题。（2）尝试右键以管理员身份运行，并查看命令行输出端的报错信息，将这些信息反馈给作者。（3）目前此程序代码虽然考虑了其他平台，但仅在Windows平台开发和反复测试过，其余操作系统运行效果未知。敬请使用MacOS和Linux的小伙伴多多测评。另提醒，请勿删除、修改_internal文件夹及其内容，它们是程序依赖的资源。

Q：显示效果不佳，有明显的错位。
A：请尝试调整窗口大小，如仍有问题，请将您的运行截图、电脑设置里的显示分辨率和缩放比反馈给作者。
Q：感觉和车内的电显不像。
A：up自己也知道：（1）过长的站名没有滚动显示；（2）绿色和橙色的那条平行四边形分界线应该上下交错；（3）下一站的橙色框应该为圆角。Up会努力做得更像，敬请期待未来版本。如果你觉得外观还有什么改进之处，或功能性建设建议，请私信up主。十分感谢！
